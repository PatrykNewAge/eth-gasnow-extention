# gasnow-extention
GasNow extension for browser

## Install

	$ yarn

## Development

    ### chrome

    window1: yarn start:chrome

    // change code
    // open new window

    window2: yarn run dev

## Build

    yarn build:chrome
    yarn build:firefox

## Environment

The build tool also defines a variable named `process.env.NODE_ENV` in your scripts.
